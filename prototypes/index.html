Proto 1

VBA Tool

Challenge:
A lot of different data from excel files have to be copied, analyzed, formatted and saved.

Solution:
One file with vba code opens all files, copies the data, analyzes, compares, changes, formats and exports the data as new file(s).

Rough details:
The whole process is splitted into pieces:
1. Open files and copy the needed data
2. Analyzing, comparing and changing the data
3. Split and format the data and save it

Proto 3

App for pathknowledge (Rails)

Challenge:
Locodrivers/office employees need to know the current situation of the pathknowledges of the locodrivers.

Solution:
Web app with login accounts for locodrivers and office employees.

Coverage of the use cases:

  1. Locodriver can create/update/delete his pathknowledges
  2. Locodriver can export a report (excel file) and send it via e-mail to his requester
  3. Locodriver can monitor the validity of the pathknowledges.
  4. Office employees can search for a pathknowledge
  5. Office employees can search for a locodriver
  6. Office employees can monitor the validity of pathknowledges
