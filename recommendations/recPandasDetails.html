How to use with jupyter

  • import pandas as pd - import the pandas module
  • df = pd.read_excel("Excelfile.xlsx") - read excel file
  • df.head() - shows the head and the first 5 rows of the file
  • df - shows the file content in a table
  • df.info() - shows the details of the data
  • df["incomes"] - shows only the data of the column
  • df["year"] != "sum:" - shows True or False
  • df2 = df[df["year"] != "sum:"] - deletes the row with "Sum:" and save the data in a new dataframe variable
  • df3 = df2.drop(8) - deletes the row 8 solution 1
  • df3 = df2[df2["year"].notnull()] - deletes the row 8 solution 2
  • profit = df3["incomes"] - df3["outcomes"] - calculate the values and save in the variable
  • ax.plot(df3["year"], profit, label="profit") - shows the profit
  • for index, row in df3.iterrows(): print row - shows the data of the data frame
  • print(row["year"]) - filters and shows the data of the rows
  • df.values - shows the data in an array
  • df.values[0] - shows the first row of the array
  • df.values[0][0] - shows the first value in the first row
  • print(len(df)) - shows the number of rows
  • print(len(df[df["year"] == 2012])) - shows the number of rows with the value 2012 for year
  • df = pd.read_excel("Excelfile.xlsx", sheet_name=["Tabelle1", "Tabelle2"]) - reads excel file with the 2 sheets and outputs a dictionary
  • df_Tabelle1 = df["Tabelle1"] - shows only the sheet Tabelle1

Example

Filter the data

import pandas as pd
df = pd.read_excel("Excelfile.xlsx")
df.head()
df
df["incomes"]
df["year"] != "sum:"
df2 = df[df["year"] != "sum:"]
df3 = df2[df2["year"].notnull()]
profit = df3["incomes"] - df3["expenses"]

Show the data

%matplotlib inline
import matplotlib.pyplot as plt
import seaborn as sns
fig, ax = plt.subplots()
fig.dpi = 100
ax.plot(df3["year"], df3["incomes"], label ="incomes")
ax.plot(df3["year"], df3["expenses"], label ="expenses")
ax.plot(df3["year"], profit, label="profit")
ax.legend()
plt.show()
