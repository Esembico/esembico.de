How to use with jupyter in anaconda

  • In Anaconda -> Environments -> Button Channels - add channel "conda-forge"
  • In Anaconda -> Environments -> left next to to button Channels - select "Not installed" and search for cartopy, activate "cartopy", click apply, click ok
  • osm_tiles = OSM() - usage of OpenStreetMap. Also available is GSM
  • plt.figure(figsize=(16, 16)) - creates new figure
  • ax = plt.axes(projection=osm_title.crs) - creates axes object
  • ax.set_extent([5.229, 15.952, 46.438, 56.097]) - creates germany
  • ax.add_image(osm_tiles, 7) - creates the osm data with the precision of 7 in a map
