How to use with jupyter in anaconda

  • In Anaconda -> Environments -> Button Channels - add channel "conda-forge"
  • In Anaconda -> Environments -> left next to to button Channels - select "Not installed" and search for cartopy, activate "cartopy", click apply, click ok
  • osm_tiles = OSM() - usage of OpenStreetMap. Also available is GSM
  • plt.figure(figsize=(16, 16)) - creates new figure
  • ax = plt.axes(projection=osm_title.crs) - creates axes object
  • ax.set_extent([5.229, 15.952, 46.438, 56.097]) - creates germany
  • ax.add_image(osm_tiles, 7) - creates the osm data with the precision of 7 in a map

df2 = df[df["reclat"].notnull()] #filters out the NaN data

long = df3["reclong"]
lat = df3["reclat"]

3. Show the relevant data in a picture/map/card

%matplotlib inline
import matplotlib.pyplot as plt
import cartopy.crs as ccrs
import cartopy.feature as cfeature

fig = plt.figure(figsize=(15, 20))
ax = plt.axes(projection=ccrs.Robinson())

ax.set_global()
ax.coastlines(resolution="50m")

ax.plot(long, lat, linestyle=" ", marker="X", transform=ccrs.PlateCarree(), color=(0, 0, 0), alpha=0.2)

ax.add_feature(cfeature.OCEAN, zorder=0)
ax.add_feature(cfeature.LAND, zorder=0)
ax.add_feature(cfeature.BORDERS, zorder=0, edgecolor='gray')
ax.gridlines()
plt.show()
