Types of Data

  • Numerical - e.g. page load times
  • Categorical - e.g. product category
  • Ordinal - e.g. movie ratings

Mean

Sum / number of samles
1 + 4 + 5 + 4 + 8 = 22 / 5 = 4.4

Median

Sort the values, take the value in the midpoint
1, 4, 4, 5, 8

Mode

The most common value in a data set
1: 1, 4: 2, 5: 1, 8: 1

Variance

Measures how "spread-out" the data is
Differences from the mean:
1 - 4.4 = - 3.4
4 - 4.4 = - 0.4
5 - 4.4 = 0.6
4 - 4.4 = - 0.4
8 - 4.4 = 3.6
Find the squared differences:
(11.56, 0.16, 0.36, 0.16, 12.96)
Find the average of the squared differences:
(11.56 + 0.16 + 0.36 + 0.16 + 12.96) / 5 = 5.04

Standard Deviation

Square root of the variance
Root of 5.04 = 2.24

Probability Density function

Gives you the probability of a data point
falling within some given range of a given value

Probabilty Mass Function

Probability of given discret values
