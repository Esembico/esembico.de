How to use with jupyter

  • %matplotlib inline - for grafics which can be saved automatically
  • %matplotlib notebook - interactive grafic
  • within interactive grafic you can: zoom, shift, scale, download and save
  • fig - figure of the whole grafic
  • ax - axes of the graph
  • dpi - resolution of the grafic
  • fig.savefig("graph_1.png") - saves grafic as image
  • fig.savefig("graph_1.png"), transpartent = True - for transparent and not white image
  • ax.set_title("Title") - titel for the grafic
  • ax.set_xlabel("xlabel") - description for the x axe
  • ax.set_ylabel("ylabel") - description for the y axe
  • ax.set_ylim([0, 10]) - sets range for the y axe
  • ax.set_xlim([0, 10]) - sets range for the x axe
  • ax.legend() - shows legend for the lines
  • ax.legend(loc=1) - shows legend in the upper right corner
  • With seaborn (library for graphics in Python) you can display better graphics
  • sns.set_style("whitegrid") - differnt views possible with white, whitegrid, dark, darkgrid
  • ax.plot(x, y, label="new customers", color="green") - changes line color with the attribute color, you can also write red as (1, 0, 0)
  • ax.set_facecolor((0.95, 0.95, 0.95)) - changes background color to grey
  • ax.plot(x, y, label="Label", linestyle=":") - sets different line styles with linestyle (":", "-.", "--")
  • ax.plot(x, y, label="Label", linewidth=4) - sets the width of the line
  • ax.plot(x, y, label="Label", marker="o") - sets a marker for the values
  • ax.plot(x, y, label="Label", marker="o", markerfacecolor=(1, 0, 0)) - sets color for the marker
  • ax.plot(x, y, label="Label", marker="o", markersize=8) - sets the size of the marker
  • ax.set_xticks([1, 3, 5]) - shows only the requested numbers
  • ax.set_xticklabels(["year 1", "year 2", "year 3"]) - shows the requested labels
  • ax.set_xticklabels(["year 1"], rotation=45) - shows the requested labels rotated
  • ax.bar(x, y) - shows bar chart
  • ax.bar(x, y, tick_label=["year 1"]) - shows label for the bars
  • ax.bar(x, y, width=0.4) - changes the width of the single bars
  • ax.barh(x, y) - shows a horizontal bar diagram
  • ax.bar(x, y, width=-0.4 align="edge") ax.bar(x, y2, width=0.4 align="edge") - shows the bars in line; makes sense for to compared bars next to it
  • ax.pie([100, 50]) - shows a pie chart
  • ax.axis("equal") - shows a precise circle
  • ax.pie([100, 50], labels=["option 1", "option 2"]) - displays the labels
  • ax.pie([100, 50], labels=["option 1", "option 2"], startangle=90) - rotates the circle
  • ax.pie([100, 50], labels=["option 1", "option 2"], explode=[0, 0, 0.1]) - shows the third piece in focus
  • ax.pie([100, 50], labels=["option 1", "option 2"], shadow=True) - shows a shadow
  • ax.pie([100, 50], labels=["option 1", "option 2"], autopct="%i") - shows the values in percent
  • ax.pie([100, 50], labels=["option 1", "option 2"], autopct="%.1f%%") - shows the comma values with percent character
  • ax.pie([100, 50], labels=["option 1", "option 2"], colors=[(1, 0, 0), (0, 1, 0)]) - define the colors for the pieces
  • ax.scatter([100, 50]) - shows a dot chart
  • fig, ax = plt.subplots(figsize=[5, 3]) - changes the size of the image and not the labels

Example

%matplotlib inline
import matplotlib.pyplot as plt
#or seaborn with import seaborn as sns
fig, ax = plt.subplots()
fig.dpi = 100
x = [1, 2, 3, 4, 5, 6, 7, 8]
y = [2, 4, 5, 3, 4, 5, 7, 2]
y2 = [4, 4, 7, 10, 4, 6, 10, 4]
ax.plot(x, y, label="new customers")
ax.plot(x, y2, label="new prospective customers")
ax.legend(loc=4)
ax.set_ylim([0, 11])
ax.set_xlim([1, 8])
ax.set_title("Performance over time")
ax.set_xlabel("Date")
ax.set_ylabel("Number")
plt.show()

Example 2 (Stacked barplot)

%matplotlib inline
import matplotlib.pyplot as plt
import seaborn as snd
fig, ax = plt.subplots()
fig.dpi = 100
x = [2011, 2013, 2015]
y = [1050, 1400, 1300] #profit product 1
y2 = [200, 100, 200] #profit product 2
ax.bar(x, y, label="profit 1", tick_label=["year 1", "year 2", "year 3"])
ax.bar(x, y2, bottom=y, label= "profit 2")
ax.legend()
ax.set_title("bar chart")
plt.show
