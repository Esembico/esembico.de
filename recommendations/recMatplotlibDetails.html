How to use with jupyter

  • %matplotlib inline - for grafics which can be saved automatically
  • %matplotlib notebook - interactive grafic
  • within interactive grafic you can: zoom, shift, scale, download and save
  • fig - figure of the whole grafic
  • ax - axes of the graph
  • dpi - resolution of the grafic
  • fig.savefig("graph_1.png") - saves grafic as image
  • fig.savefig("graph_1.png"), transpartent = True - for transparent and not white image
  • ax.set_title("Title") - titel for the grafic
  • ax.set_xlabel("xlabel") - description for the x axe
  • ax.set_ylabel("ylabel") - description for the y axe
  • ax.set_ylim([0, 10]) - set range for the y axe
  • ax.set_xlim([0, 10]) - set range for the x axe
  • ax.legend() - show legend for the lines
  • ax.legend(loc=1) - show legend in the upper right corner
  • With seaborn (library for graphics in Python) you can display better graphics
  • sns.set_style("whitegrid") - differnt views possible with white, whitegrid, dark, darkgrid

Example

%matplotlib inline
import matplotlib.pyplot as plt
#or seaborn with import seaborn as sns
fig, ax = plt.subplots()
fig.dpi = 100
x = [1, 2, 3, 4, 5, 6, 7, 8]
y = [2, 4, 5, 3, 4, 5, 7, 2]
y2 = [4, 4, 7, 10, 4, 6, 10, 4]
ax.plot(x, y, label="new customers")
ax.plot(x, y2, label="new prospective customers")
ax.legend(loc=4)
ax.set_ylim([0, 11])
ax.set_xlim([1, 8])
ax.set_title("Performance over time")
ax.set_xlabel("Date")
ax.set_ylabel("Number")
plt.show()
