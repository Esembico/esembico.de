Lifecycle

  1. Capture: Data acquisition, data entry, signal reception, data extraction
  2. Maintain: Data warehousing, data cleansing, data staging, data processing, data architecture
  3. Process: Data mining, clustering/classification, data modeling, data summarization
  4. Communicate: Data reporting, data visualization, business intelligence, decision making
  5. Analyze: Exploratory/confirmatory, predictive analysis, regression, text mining, qualitative analysis

Components

  1. Statistics: Statistics is the most critical unit of Data Science basics. It is the method or science of collecting and analyzing numerical data in large quantities to get useful insights.
  2. Data engineering: involves acquiring, storing, retrieving, and transforming the data. Data engineering also includes metadata (data about data) to the data.
  3. Machine learning: Machine Learning explores the building and study of algorithms which learn to make predictions about unforeseen/future data.
  4. Visualization: Data visualization is meant by representing data in a visual context so that people can easily understand the significance of data. Data visualization makes it easy to access the huge amount of data in visuals.
  5. Advanced computing: Heavy lifting of data science is advanced computing. Advanced computing involves designing, writing, debugging, and maintaining the source code of computer programs.
  6. Deep Learning: Deep Learning method is new machine learning research where the algorithm selects the analysis model to follow

Applications

  • Prediction (predict a value based on inputs)
  • Classification (e.g., spam or not spam)
  • Recommendations (e.g., Amazon and Netflix recommendations)
  • Pattern detection and grouping (e.g., classification without known classes)
  • Anomaly detection (e.g., fraud detection)
  • Recognition (image, text, audio, video, facial, …)
  • Actionable insights (via dashboards, reports, visualizations)
  • Automated processes and decision-making (e.g., credit card approval)
  • Scoring and ranking (e.g., FICO score)
  • Segmentation (e.g., demographic-based marketing)
  • Optimization (e.g., risk management)
  • Forecasts (e.g., sales and revenue)

Tools

  • Data Analysis tools: R, Python, Statistics, SAS/ACCESS, Jupyter, R Studio, MATLAB, Excel, RapidMiner, Julia
  • Data Warehousing: ETL, SQL, Hadoop, Informatica/Talend, AWS Redshift
  • Data Visualization tools: R, Jupyter, Tableau, Cognos, Qlick, google charts, PowerBI, Webix
  • Machine learning tools: Spark, Mahout, Azure ML studio.

Roles

  1. Data Scientist
    A data scientist is responsible to collect a big dump of data, study it, sift relevant data, and use tools like SAS, R programming, and Python, etc. to extract insights that can be used to improve the organization’s efficiency and output. Tools : R, SAS, Python, SQL, Hive, Matlab, Pig, Spark
  2. Data Engineer
    The role of data engineer is of working with large amounts of data. He develops, constructs, tests, and maintains architectures like large scale processing system and databases. Tools: SQL, Hive, R, SAS, Matlab, Python, Java, Ruby, C + +, and Perl
  3. Data Analyst
    A Data Analyst works with project managers to identify critical metrics and KPIs, and deliver actionable insights to relevant decision-makers. Like the data scientist, the analyst also performs statistical analysis and make predictive models Tools: R, Python, HTML, JS, C, C+ + , SQL
  4. Statistician
    A statistician is responsible for creating data-driven surveys, opinion polls, and questionnaires. Tools: SQL, R, Matlab, Tableau, Python, Perl, Spark, and Hive
  5. Data Architect
    The role of a Data Architect is to integrate, centralize, protect, and maintain the data sources of an organization. Tools: SQL, XML, Hive, Pig, spark
  6. Data Admin
    Data admin should ensure that the database is accessible to all relevant users. He also makes sure that it is performing correctly and also ensures security. Tools: Ruby on Rails, SQL, Java, C#, and Python
  7. Data/Analytics Manager
    The Data and Analytics Manager assumes the role of assigning duties and operations to the data science team Tools : SQL, R, SAS, python, Matlab, Java
